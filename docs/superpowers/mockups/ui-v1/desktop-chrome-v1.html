What's in this mockup (5 areas — call out what stays for v1)

1

Top bar — brand + student

Dark header with app name and student name/avatar on the right. Minimal but signals "this is your account."

2

Flow switcher (sidebar) — Active + Pending cards

Vertical list of all the student's StudentDegreeFlows. Click to switch which one is rendered in the main area. Active flows have solid borders; Pending (what-if) have dashed borders. Includes "+ Explore another flow" link.

3

Progress stats panel (sidebar)

Credit progress bar (28 / 125), counts by status (5 completed / 6 in progress / 22 planned / 2 warnings), projected graduation term.

4

Issues panel (sidebar) — validation findings

List of current ValidationIssues for the active flow. Cascaded moves, semester overloads, broken pairings. Each clickable to jump to the relevant tile.

5

Action buttons (main header)

Print / Export / Edit buttons in the top-right of the main area. v1 might be read-only (no Edit) or include the Edit affordance even if it's stubbed.

Tip: select multiple. Anything not selected gets deferred to v2 (or considered for cut entirely).
