# ISUCourseManager.Web

React + Vite + TypeScript SPA for the ISU Course Manager. Talks to the ASP.NET API in `src/ISUCourseManager.Api/` at `/api/v1/`.

## Commands

```
npm install         # one-time setup
npm run dev         # dev server with HMR at http://localhost:5173
npm run build       # type-check + production build to dist/
npm run preview     # serve the production build locally
npm run lint        # ESLint
```

## Status

UI v1 in progress, scaffold-and-iterate. Real screens land step by step; see the branch history.
